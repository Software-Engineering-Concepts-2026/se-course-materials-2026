### **Required Readings / Book References**

- **[ESP] Engineering Software Products, An Introduction to Modern Software Engineering by Ian Sommerville**
    - **Chapter 1: Software Products:** อ่านทั้งบทเพื่อทำความเข้าใจความแตกต่างพื้นฐานระหว่าง Project-based และ Product-based software engineering, Product Vision, และ Software Execution Models (Stand-alone, Hybrid, Software as a service)
- **[PP] The Pragmatic Programmer, your journey to mastery by David Thomas, Andrew Hunt**
    - **Chapter 1: A Pragmatic Philosophy:** อ่านทั้งหมดเพื่อซึมซับปรัชญาหลักของหลักสูตร
        - **Topic 1: It's Your Life** (เน้นเรื่อง Agency และความรับผิดชอบต่อการเรียนรู้)
        - **Topic 2: The Cat Ate My Source Code** (เน้นเรื่องการเป็นเจ้าของผลงานและไม่หาข้ออ้าง)
        - **Topic 3: Software Entropy** (ทำความเข้าใจทฤษฎี Broken Window และความสำคัญของการรักษาคุณภาพโค้ด)
